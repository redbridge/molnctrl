=========================================
molnctrl - A Cloudstack python API client
=========================================

This is molnctrl a simple python Cloudstack API library.

Usage::

    import molnctrl
    csapi = molnctrl.Initialize("apikey", "api_secret", api_host='cloud.fqdn')
    accounts = csapi.list_accounts()
